# Assets

Drop files here and tell Claude — they'll get wired into the pages.

## Folders

- `fonts/` — custom fonts. Preferred format: **WOFF2** (smallest, all modern browsers). If you only have TTF/OTF, drop them anyway and they can be converted.
- `icons/` — icons and logos. Preferred format: **SVG** (scales sharp, tiny, can be recoloured with CSS). PNG works too.
- `img/` — photos and images (workshop, team, hero shots). Any format is fine to drop (JPG/PNG/HEIC); they'll be converted to **WebP/AVIF** and lazy-loaded to keep the site under the 3-second mobile load target from the blueprint.

## Naming

No strict rules — descriptive names help, e.g. `workshop-exterior.jpg`, `logo-full.svg`, `heading-font.woff2`.
